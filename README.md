# Rust Workspace Example
# Usage

## Development

### Add A New Service

    make new-app name=<your_service_name>

Example

    make new-app name=my_service

### Add A New Library

    make new-lib name=<your_lib_name>

Example

    make new-lib name=my_lib

### Run Service

    make run name=<your_service_name>

Example

    make run name=my_service
